# JavaProjects – Rendus des Prosits Java

## Description
Ce dépôt contient l’ensemble de mes rendus de prosits réalisés dans le cadre du cours de programmation Java.  
Les travaux sont organisés exclusivement à travers des branches et des commits afin d’assurer une structure propre et conforme aux bonnes pratiques Git.

## Organisation du dépôt
- Chaque prosit est associé à une branche dédiée.
- Les commits décrivent les différentes étapes de réalisation.
- La branche principale sert de référence générale.

## Utilisation
Pour consulter un rendu :
- Se positionner sur la branche correspondante.
- Chaque branche contient le code Java relatif au prosit traité.

L’exécution des programmes peut se faire via un IDE Java ou par ligne de commande après compilation.

## Technologies
- Java (JDK)
- Git / GitHub
- IDE Java (au choix)

## Objectif
Ce dépôt a pour objectif de centraliser les rendus des prosits, d'assurer leur traçabilité et de maintenir une organisation professionnelle.
